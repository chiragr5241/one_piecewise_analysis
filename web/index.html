About This Analysis

This visualization recreates the analysis from Not David's video on anime character networks. Characters are connected if they appear in the same episode. The friendship score between two characters is:

Fij = Nij / max(Ni, Nj)

where Nij is episodes they share, and Ni, Nj are their total episode counts. Data scraped from the One Piece Wiki using the MediaWiki API.
